NPV Calculation (Python)


def npv(cashflows, r):
    total = 0
    for t, cf in enumerate(cashflows):
        total += cf / ((1+r)**t)
    return total

Financial Formula

$$NPV = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t}$$

Python Code Example


def npv(cashflows, rate):
    total = 0
    for t, cf in enumerate(cashflows):
        total += cf / ((1 + rate) ** t)
    return total

print(npv([-1000, 500, 600, 700], 0.08))

Financial Formulas

Net Present Value (NPV): $$NPV = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t}$$

Internal Rate of Return (IRR): $$0 = \sum_{t=0}^{T} \frac{C_t}{(1+IRR)^t}$$
